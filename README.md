# About

This is solo indie game built using [Godot](https://godotengine.org/). It is a game about popping bubble wrap, getting views, and building your bubble popping empire. Inspired by games like [Dice a Million](https://store.steampowered.com/app/3430340/Dice_A_Million/) and [Cookie Clicker](https://cookieclicker.com/), this game combines rougelites themes with gameplay similar to idle games.

## Roadmap
As a solo dev going back to school for my masters degree, I do not have a difinitive timeline, so instead here is a list of some features I am looking to include:
- Hand Customization (Nails, Skin Tone, Scars or other Markings)
- Different Sizes of Bubble Wrap Sheets
- Holidays (Every Holiday will have custom accessories and bubble wrap sheets that boost views)
- Rings, Bracelets, and Decorations that have different effects on views
- Brand Deals (Careful taking too many; viewers may not approve)
- Multiple Endings

## THIS IS A WORK IN PROGRESS
While I am excited to work on this project, it is still very early in development. I have not even come up with a name yet, so many parts of this are subject to change. I am excited to share more details as I progress!
